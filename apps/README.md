# SAM 2 Video Gradio App

本目录提供一个轻量本地网页，用于交互式视频分割：

- 在页面里浏览并选择视频目录，扫描其中的视频文件或 JPEG 帧目录
- 在任意帧上用正点、负点、包围盒提示对象
- 支持多对象标注
- 支持同一个对象在多个帧上补充标注，并向前/向后传播
- 对象标签和统计信息显示在图像外，标注图内只显示点和框
- 支持按 task 管理标注任务：人工标注直接保存到 `<输出根目录>/<task>/<video_name>_sam2_masks/sam2_prompts.json`
- 每个 task 的类别由 `<输出根目录>/<task>/labels.txt` 维护，新增对象时只能从该 task 的标签列表选择
- 标注显示可切换为当前对象、全部显示或隐藏标注
- 新增对象后需点击“完成当前对象”，才能继续新增下一个对象
- 点击“新增对象”会自动切换到框标注模式
- 已完成对象可在“当前对象”中重新选中并继续补充点或框
- 已完成对象不在当前帧时，可在“全部已有对象（跨帧继续标注）”中选回同一个实例并在当前帧补充标注
- 已有对象的类别可通过“修改当前对象类别”显式更新，单纯切换“对象类别”不会改动已完成对象
- 每个对象拥有唯一可编辑颜色；mask 叠加透明度可手动调整
- 如果 `<输出根目录>/<task>/<video_name>_sam2_masks/mask_frames/` 已存在，可在标注时叠加显示已有传播 mask
- 如果 `<输出根目录>/<task>/<video_name>_sam2_masks/overlay.mp4` 已存在，加载视频时会自动显示到 Overlay 视频播放器
- 标注画布带有图像外边距，框选边缘物体时可以点到图像外，保存时会自动裁剪到图像边界
- 可选择 SAM 2.1 tiny/small/base+/large checkpoint
- 一键传播到全视频，并导出 overlay 帧、类别 ID mask、彩色 mask、单对象二值 mask 和 overlay MP4
- 传播完成后会直接把当前帧 mask 叠加回标注窗口，不再单独显示第一帧预览
- 每次传播结束会释放 SAM2 视频状态和 CUDA cache；也可手动点击“卸载模型并释放显存”
- 默认启用视频帧和状态 CPU offload，并启用 SAM2 的 non-conditioning memory 清理以降低多帧 prompt 的显存峰值

## 安装

先按仓库 README 安装 SAM 2、PyTorch 和 checkpoints，然后安装网页依赖：

```bash
pip install -e ".[notebooks]" gradio decord
```

普通视频文件会优先用系统 `ffmpeg` 抽帧；如果没有 `ffmpeg`，会自动回退到 `decord` 抽帧。

## 运行

在仓库根目录执行：

```bash
python apps/sam2_video_gradio_app.py
```

浏览器打开：

```text
http://localhost:7860
```

## 使用流程

1. 在“子目录”中选择目录，点击“进入”或“上一级”浏览到视频目录。
2. 点击“使用当前目录”或“扫描视频”。
3. 在“输出根目录”中设置任务根目录，例如 `outputs/`，点击“扫描 task”。
4. 选择或输入 task 名称，点击“使用 task”。程序会使用或创建 `<输出根目录>/<task>/`。
5. task 的类别在 `<输出根目录>/<task>/labels.txt` 中维护；可在“新增标签”中写入新类别。
6. 选择视频或 JPEG 帧目录，点击“加载视频”。程序会按视频名自动加载 `<输出根目录>/<task>/<video_name>_sam2_masks/sam2_prompts.json`。
7. 用“当前标注帧”滑块选择要标注的帧。
8. 需要更细标注时，用“标注放大倍数”放大图像，不建议使用浏览器或 Gradio 图片自带全屏缩放。
9. 在“标注操作”中从“对象类别”选择 task 定义好的标签，点击“新增对象”。程序会默认切到“框”模式，也可再改为正点或负点。
10. 一个对象标注完后点击“完成当前对象”，之后才能新增下一个对象。
11. 已完成对象需要在同一帧补充标注时，在“当前对象”中选回该对象，直接继续点图像即可重新编辑。
12. 已完成对象需要在其他帧补充标注时，切到目标帧，在“全部已有对象（跨帧继续标注）”中选回同一个对象，再继续加点或框。
13. 如果完成上一个对象后已切换到新的对象类别，直接在图像上画框会自动新建该类别的对象。
14. 如果需要修改已有对象类别，先在“当前对象”中选中对象，再选择新的“对象类别”，点击“修改当前对象类别”。
15. 如果需要修改对象颜色，先选中对象，在“当前对象颜色”选择新颜色后点击“应用对象颜色”。已被其他对象使用的颜色会被拒绝。
16. 用“mask 透明度”调整传播 mask 和 overlay 的透明度。
17. 用“标注显示”选择当前对象、全部显示或隐藏标注，避免遮挡其他标注。
18. 可用“显示已有传播 mask”叠加或隐藏上一次生成的 mask。
19. 框模式需要点击两次，分别作为矩形的两个角；边缘物体可以把点点到图像外灰色边距里，保存时会自动取边界极限值。
20. 多个类别/实例可以重复新增对象并分别标注；物体不在第一帧时，切到出现它的帧再标注。
21. 点击“开始分割并导出”。如果连续运行时显存紧张，保持 CPU offload 开启；仍不足时可改用 Base+/Small，或点击“卸载模型并释放显存”后再运行。

结果默认写到：

```text
outputs/<task>/<video_name>_sam2_masks/
```

其中：

- `sam2_prompts.json` 是人工交互标注文件，和 mask 结果放在同一个视频 mask 目录下。
- `mask_frames/` 是类别 ID 图，像素值 `0` 表示背景，`1/2/...` 表示对象编号，直接用图片查看器打开会接近黑色。
- `mask_visual_frames/` 是彩色可视化 mask，颜色来自页面里每个对象的颜色设置。
- `object_mask_frames/` 是每个对象单独的 `0/255` 二值 mask，文件名格式为 `<frame>_obj<object_id>.png`，白色表示该对象，黑色表示背景。
- `overlay_frames/` 和 `overlay.mp4` 是叠加到原视频上的可视化结果。

task 标签文件格式是纯文本，每行一个类别名，例如：

```text
needle
gripper
tissue
occluder
```
